# Choreography Editor

  Aplikacija za sinkronizaciju koreografija sinkroniziranog plivanja.

  Vibracijski obrasci za 4 grupe pametnih satova (G1-G4).

  ## Stack
  - React 18 + TypeScript
  - Node.js + Express
  - PostgreSQL + Drizzle ORM
  - Tailwind CSS v4
  - shadcn/ui

  ## Pokretanje

  ```bash
  npm install
  npm run dev
  ```
